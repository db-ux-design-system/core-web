Adding Formatted Infotextadding-formatted-infotext anchor link

To add a descriptive text with HTML formatting (e.g., bold or italic text) to a checkbox, you should use a separate element for the message and link it via the aria-describedby-HTML-attribute for accessibility. This ensures that screen readers correctly associate the message with the checkbox.

<!-- Checkbox with formatted infotext -->
...
<body>
	<label class="db-checkbox" for="checkbox-element">
		<input
			type="checkbox"
			id="checkbox-element"
			name="States"
			aria-describedby="checkbox-message"
		/>
		Label
	</label>
	<p class="db-infotext" id="checkbox-message">
		Example <strong>Text</strong>
	</p>
</body>
