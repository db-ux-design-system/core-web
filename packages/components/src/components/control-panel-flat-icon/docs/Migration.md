## DB UX v4 ➡ DB UX v5

`DBControlPanelFlatIcon` is a new component introduced in v5. It provides an optional flat icon navigation for a mobile app look, visible when using `variant="flat-icon"` on the control panel.

This is a new component with no direct predecessor in v4.
