## React

For general installation and configuration take a look at the [react-components](https://www.npmjs.com/package/@db-ui/react-components) package.

### Use component

```tsx App.tsx
// App.tsx
import { DBMultiSelectHeader } from "@db-ui/react-components";

const App = () => (
	<DBMultiSelectHeader>
		MultiSelectHeader
	</DBMultiSelectHeader>
);

export default App;
```
